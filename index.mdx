  <h2 style={{fontSize: "30px", fontWeight: "700", color: "#ffffff", fontFamily: "Inter, sans-serif", textAlign: "center", marginBottom: "12px"}}>From zero to working mod in minutes</h2>
  <p style={{textAlign: "center", color: "#848c8e", fontFamily: "Inter, sans-serif", marginBottom: "36px"}}>Use the SDK's high-level builders — then call <code>mod.build()</code> to generate all XML and `.modinfo`.</p>

```typescript quickstart.ts
import { Mod, UnitBuilder, ACTION_GROUP_BUNDLE, UNIT_CLASS, UNIT } from '@mateicanavra/civ7-sdk';

const mod = new Mod({
  id: 'my-first-mod',
  name: 'My First Mod',
  version: '1.0.0',
});

const unit = new UnitBuilder({
  actionGroupBundle: ACTION_GROUP_BUNDLE.AGE_ANTIQUITY,
  typeTags: [UNIT_CLASS.RECON, UNIT_CLASS.RECON_ABILITIES],
  unit: {
    unitType: 'UNIT_CUSTOM_SCOUT',
    baseMoves: 2,
    baseSightRange: 10,
  },
  unitCost: { cost: 20 },
  unitStat: { combat: 0 },
  unitReplace: { replacesUnitType: UNIT.SCOUT },
  visualRemap: { to: UNIT.ARMY_COMMANDER },
  localizations: [
    { name: 'Custom Scout', description: 'An enhanced recon unit' }
  ],
});

mod.add([unit]).build('./my-mod');
```
