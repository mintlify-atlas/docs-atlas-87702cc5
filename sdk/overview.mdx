The `@mateicanavra/civ7-sdk` package lets you write Civilization VII mods entirely in TypeScript. Instead of editing XML files by hand, you instantiate builders, configure them with typed options, and call `mod.build()` to generate a complete mod directory â€” including the `.modinfo` file and all XML.

## Two APIs

The SDK exposes two layers. You can use either one, or mix them in the same mod.

## Installation

The SDK is published to the GitHub Package Registry. Before installing, add the registry entry to your project's `.npmrc`:

```bash .npmrc
@mateicanavra:registry=https://npm.pkg.github.com/
```

Then install:

```bash
bun add @mateicanavra/civ7-sdk
```

<Note>
You need a GitHub personal access token with `read:packages` scope to install from the GitHub Package Registry. Set it as `NPM_TOKEN` or configure it in your `.npmrc`. See [GitHub docs](https://docs.github.com/en/packages/working-with-a-github-packages-registry/working-with-the-npm-registry#authenticating-to-github-packages) for details.
</Note>

## Quick example

```typescript
import { Mod, UnitBuilder, ACTION_GROUP_BUNDLE, UNIT_CLASS, UNIT } from '@mateicanavra/civ7-sdk';

const mod = new Mod({
    id: 'my-first-mod',
    version: '1.0.0',
});

const unit = new UnitBuilder({
    actionGroupBundle: ACTION_GROUP_BUNDLE.AGE_ANTIQUITY,
    typeTags: [UNIT_CLASS.RECON, UNIT_CLASS.RECON_ABILITIES],
    unit: {
        unitType: 'UNIT_CUSTOM_SCOUT',
        baseMoves: 2,
        baseSightRange: 10,
    },
    unitCost: { cost: 20 },
    unitStat: { combat: 0 },
    unitReplace: { replacesUnitType: UNIT.SCOUT },
    visualRemap: { to: UNIT.ARMY_COMMANDER },
    localizations: [
        { name: 'Custom Scout', description: 'An enhanced recon unit' }
    ],
});

mod.add([unit]).build('./my-mod');
```

Running this script writes `my-mod/my-first-mod.modinfo` plus all required XML files under `my-mod/units/unit-custom-scout/`.
