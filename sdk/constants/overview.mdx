---

The SDK exports typed constant objects for all standard game identifiers. Import them from the package root:

```typescript
import {
    UNIT,
    UNIT_CLASS,
    ABILITY,
    EFFECT,
    COLLECTION,
    REQUIREMENT,
    KIND,
    AGE,
    TERRAIN,
    BIOME,
    RESOURCE,
    YIELD,
    DISTRICT,
    FEATURE,
    TRAIT,
    TAG_TRAIT,
    ACTION_GROUP,
    ACTION_GROUP_ACTION,
    ACTION_GROUP_BUNDLE,
} from '@mateicanavra/civ7-sdk';
```

All constants use `SCREAMING_SNAKE_CASE` and are defined `as const`, giving you precise literal types and full IntelliSense in any TypeScript editor.

|---|---|---|
| `UNIT` | `SCOUT`, `ARCHER`, `SETTLER`, `ARMY_COMMANDER` | `unitReplace`, `visualRemap` |
