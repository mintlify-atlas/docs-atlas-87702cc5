This guide gets you from zero to a working mod. You will install the project dependencies, link the CLI globally, pull game data from your Civilization VII installation, and build a custom unit using the SDK.

## Prerequisites

Before you start, make sure you have:

- **Node.js 22.14 or later** — check with `node --version`
- **Bun** — install from [bun.sh](https://bun.sh)
- **Civilization VII** installed locally (required for the game data step)

## Steps

<Steps>
  <Step title="Clone the repository">
    Clone the monorepo and navigate into it:

    ```bash
    git clone https://github.com/mateicanavra/civ7-modding-tools.git
    cd civ7-modding-tools
    ```
  </Step>

  <Step title="Install dependencies and link the CLI">
    Run both commands from the repo root. The first installs all workspace dependencies using the lockfile. The second builds the CLI and links the `civ7` binary globally so you can run it from any directory.

    ```bash
    bun install --frozen-lockfile
    bun run link:cli
    ```

    <Note>
      After linking, you may need to open a new terminal or source your shell profile before `civ7` becomes available. Run `civ7 --help` to verify the installation.
    </Note>
  </Step>

  <Step title="Configure your Civ7 install path">
    Open `civ.config.jsonc` in the repo root and set `inputs.installDir` to the path of your Civilization VII installation:

    ```jsonc
    {
        "inputs": {
            "installDir": "~/Library/Application Support/Steam/steamapps/common/Sid Meier's Civilization VII/CivilizationVII.app/Contents/Resources"
        },
        "outputs": {
            "baseDir": ".civ7/outputs",
            "zip": {
                "dir": "archives",
                "name": "civ7-official-resources.zip"
            },
            "unzip": {
                "dir": "resources"
            }
        }
    }
    ```

    See [Configuration](/setup/configuration) for the full list of options and platform-specific install paths.
  </Step>

  <Step title="Pull game data">
    This command runs `civ7 zip` followed by `civ7 unzip`, placing the extracted resources in `.civ7/outputs/resources/`:

    ```bash
    bun run refresh:data
    ```

    You only need to run this once. The `docs` and `playground` apps sync what they need from this central location automatically when you start them.
  </Step>

  <Step title="Install the SDK in your mod project">
    Create a new directory for your mod (or use an existing one) and install the SDK:

    ```bash
    mkdir my-first-mod && cd my-first-mod
    bun init -y
    bun add @mateicanavra/civ7-sdk
    ```
  </Step>

  <Step title="Create your first mod">
    Create a file called `mod.ts` and add the following code. This creates a custom scout unit that replaces the base game's `UNIT_SCOUT` and uses the army commander's visual model:

    ```typescript mod.ts
    import { Mod, UnitBuilder, ACTION_GROUP_BUNDLE, UNIT_CLASS, UNIT } from '@mateicanavra/civ7-sdk';

    const mod = new Mod({
        id: 'my-first-mod',
        version: '1.0.0',
    });

    const unit = new UnitBuilder({
        actionGroupBundle: ACTION_GROUP_BUNDLE.AGE_ANTIQUITY,
        typeTags: [UNIT_CLASS.RECON, UNIT_CLASS.RECON_ABILITIES],
        unit: {
            unitType: 'UNIT_CUSTOM_SCOUT',
            baseMoves: 2,
            baseSightRange: 10,
        },
        unitCost: { cost: 20 },
        unitStat: { combat: 0 },
        unitReplace: { replacesUnitType: UNIT.SCOUT },
        visualRemap: { to: UNIT.ARMY_COMMANDER },
        localizations: [
            { name: 'Custom Scout', description: 'An enhanced scout unit' }
        ],
    });

    mod.add([unit]).build('./output');
    ```
  </Step>

  <Step title="Build and inspect the output">
    Run the script to generate your mod files:

    ```bash
    bun run mod.ts
    ```

    The SDK creates the `output/` directory containing your `.modinfo` file and all generated XML:

    ```
    output/
    ├── my-first-mod.modinfo
    └── units/
        └── UNIT_CUSTOM_SCOUT/
            ├── database.xml
            └── localizations.xml
    ```

    Copy the `output/` folder into your Civilization VII mods directory to load it in-game.
  </Step>
</Steps>
